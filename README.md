faye-rack-example
=================

A very simple example of using Faye with Rack as a server, and a couple of clients - one in Ruby and one in JavaScript (in HTML)
  
* rackup -E production
* ruby listener.rb
* open client.html
* ruby sender.rb
  
Done!
